 COVID-19 Data Analysis & Dashboard Project

## Project Overview
This project focuses on the comprehensive analysis of COVID-19 data across various states and union territories. Utilizing Microsoft Excel, the project aims to uncover critical trends, patterns, and insights regarding the spread, public health impact, and vaccination progress of the pandemic. The final deliverable includes an interactive dashboard designed to facilitate data-driven decision-making through clear visual communication.

## Dataset Description & Source
The analysis is built upon a detailed dataset containing state-level pandemic metrics:
* *Key Metrics:* Confirmed cases, active cases, recovered cases, and mortality figures.
* *Vaccination Data:* Tracking progress for both first doses (Dose 1) and second doses (Dose 2).
* *Data Source:* Publicly available data compiled from prsindia.org.

## Methodology & Key Features
The project followed a structured data analysis workflow within Microsoft Excel:
* *Data Cleaning & Preparation:* Processing the raw dataset to ensure accuracy, consistency, and readiness for analysis.
* *Data Summarization:* Leveraging Excel formulas and Pivot Tables to aggregate data and compute critical Key Performance Indicators (KPIs).
* *Interactive Dashboard Development:* Designing a user-friendly, dynamic dashboard that integrates Pivot Charts and advanced filtering mechanisms (such as slicers) to allow users to drill down into state-specific statistics.

## Key Insights Addressed
* Comparative analysis of COVID-19 impact across different states and union territories.
* Visual trends highlighting the progression of active cases relative to recoveries and deaths.
* Evaluation of vaccination deployment and coverage progress nationwide.

## Core Competencies Demonstrated
* Data analysis and statistical aggregation.
* Data visualization and interactive dashboard design.
* Business intelligence techniques using Microsoft Excel.
* Interpretation of large-scale public health datasets to support analytical reporting.
